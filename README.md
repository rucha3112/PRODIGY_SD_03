Contact Management System
Overview

The Contact Management System is a web-based application that allows users to store and manage contact information efficiently. Users can add, edit, delete, and search contacts. The application uses Local Storage to save data, ensuring contacts remain available even after refreshing the browser.

Features
Add new contacts
View contact list
Edit existing contacts
Delete contacts
Search contacts instantly
Persistent data storage using Local Storage
User-friendly interface
Technologies Used
HTML5
CSS3
JavaScript
Local Storage API
How to Use
Enter the contact's Name, Phone Number, and Email Address.
Click the Add Contact button to save the contact.
View all contacts in the table below.
Click Edit to modify a contact's details.
Click Delete to remove a contact.
Use the search box to quickly find contacts.
